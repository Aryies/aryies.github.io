Unit 1 Outcome 3 School Assessed Coursework Assessment Task

Instructions:
Working in teams, you are to use web-authoring software to create a website that presents an overview of an issue associated with one context. With evidence acquired from primary and secondary sources you should present:

  • the viewpoints of different stakeholders
  • your team’s viewpoint regarding the issue
  • strategies suggested for encouraging individuals to inﬂuence how ICT is applied in this context to solve or lessen the impact of the issue.

You should use spreadsheet software to manipulate acquired primary data and generate graphical representations to include in your website.

You are able to supplement these graphical representation with related sound and/or image ﬁles.

You are to use visualising thinking tools and techniques to assist in formulating the teams opinions which should also should be included on the website.

Project plans need to be developed using ICT to record tasks to be completed and note team-member responsibilities. The progress of the team is to be documented on the tool you have developed for this purpose.

The SAC requires you to apply all the stages of the PSM.

You have 6 classes to work on and complete this SAC.

Expectations
This SAC is marked out of 50 with all stages of the PSM expected to be addressed. Check the assessment rubric for a more detailed description of where and how the marks are allocated.
